- **Why:** The site uses GTranslate for client-side translation but had no hreflang tags. While GTranslate is JS-based (not SEO-friendly for translated URLs), hreflang tags signal to Google which language versions exist and prevent duplicate content issues.
- **Expected Impact:** MEDIUM — Prevents potential duplicate content flags from Google for translated content. German added to match target market.
