### Core Web Vitals Optimization Checklist

```
LCP (Largest Contentful Paint) — Target < 2.5s
□ Hero image: WebP format, lazy loading OFF (above fold), preload tag
□ <link rel="preload" as="image" href="hero.webp"> in <head>
□ Use CDN (Cloudflare, BunnyCDN) — serve from closest edge node
□ Font: preload critical fonts, use font-display: swap
□ Server response time < 200ms (use caching: Redis, Varnish)

INP (Interaction to Next Paint) — Target < 200ms
□ Defer non-critical JavaScript
□ Break up long tasks (>50ms) into smaller chunks
□ Avoid large DOM size (< 1500 nodes)
□ Debounce scroll/resize listeners
□ Use Intersection Observer for lazy content

CLS (Cumulative Layout Shift) — Target < 0.1
□ Always define image width + height attributes
□ Reserve space for ads / embeds
□ Avoid inserting dynamic content above existing content
□ Use CSS transform for animations (not top/left/margin)
□ Load web fonts with size-adjust to prevent FOUT
```

### Site Speed Stack Recommendation
```
Hosting:          Kinsta / WP Engine / Cloudways (Managed WordPress)
CDN:              Cloudflare Pro (or BunnyCDN for images)
Caching:          WP Rocket or LiteSpeed Cache
Image Opt:        ShortPixel (WebP conversion) + lazy loading
Code Minify:      Autoptimize or WP Rocket
Database Opt:     WP-Optimize
Monitoring:       Google Search Console + PageSpeed Insights + GTmetrix
```
